# Student Portfolio Website (HTML + CSS)

## Project Overview
This is a multi-page **Student Portfolio Website** built using only **HTML** and **basic CSS**.  


## Features
- 4 separate HTML pages:
  - `index.html` → Home page
  - `about.html` → About Me page
  - `skills.html` → Skills & Projects page
  - `contact.html` → Contact Form page
- Common navigation bar on all pages (Home, About, Skills, Contact)[web:24]
- Uses **float** for navbar and About page layout
- Uses **Flexbox** for Skills / Projects cards layout[web:18]
- Styled HTML contact form with Name, Email, Subject, and Message fields[web:27]
- Clean folder structure suitable for GitHub and assignments

## Technologies Used
- HTML5 (semantic structure)
- CSS3 (basic styling, float, flexbox).
